📋 必須要件

【オペレーティングシステム】
  ✓ Windows 10 (64-bit) Build 1809 以上
  ✓ Windows 11 (64-bit)

【プロセッサ】
  ✓ Intel Core i3 以上（または同等のAMD）
  ✓ 1.6 GHz以上

【メモリ（RAM）】
  ✓ 最小: 4GB
  ✓ 推奨: 8GB以上

【ストレージ】
  ✓ 最小: 500MB の空き容量
  ✓ 推奨: 1GB以上（データ保存用）

【ディスプレイ】
  ✓ 最小解像度: 1024x768
  ✓ 推奨解像度: 1920x1080（Full HD）

【インターネット接続】
  ✓ 初回インストール時のみ必要
  ✓ 以降はオフラインで使用可能
  ✓ アップデート時に必要

【その他】
  ✓ 管理者権限（インストール時のみ）
  ✓ .NET Framework 4.7.2以上（通常は自動インストール）

Q: Windows 7でも使えますか？
A: いいえ。Windows 10以上が必要です。

Q: 32bit版Windowsで使えますか？
A: いいえ。64bit版のみ対応しています。

Q: メモリが4GB未満でも使えますか？
A: 動作する可能性はありますが、推奨しません。

Q: タブレットで使えますか？
A: Windows タブレットなら使用可能です。
   iPad/Androidの場合はPWA版をご利用ください。

Q: オフラインで使えますか？
A: はい。初回インストール後はオフラインで使用できます。
