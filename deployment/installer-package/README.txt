📦 このパッケージについて
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このパッケージには、あおば給食摂食量管理アプリのインストーラーが
含まれています。

📄 含まれるファイル
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ あおば給食管理-Setup-2.1.0.exe    - インストーラー
  ✅ インストールガイド.pdf             - 詳細なインストール手順
  ✅ システム要件.txt                   - 必要な環境
  ✅ README.txt                         - このファイル
  ✅ お問い合わせ.txt                   - サポート情報

💻 システム要件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  OS:       Windows 10 (64-bit) 以上
  メモリ:   4GB以上（8GB推奨）
  ディスク: 500MB以上の空き容量
  ネット:   初回のみ必要（オフライン使用可）

🚀 クイックスタート
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 「あおば給食管理-Setup-2.1.0.exe」をダブルクリック
2. インストールウィザードに従って進む
3. 完了後、デスクトップのショートカットから起動

⚠️ 初回実行時の警告について
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows Defender SmartScreen の警告が表示される場合:

  1. 「詳細情報」をクリック
  2. 「実行」をクリック

これはアプリが未署名のため表示されますが、安全性に問題はありません。

✨ 主要機能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✓ 摂食量記録（1割～10割の10段階）
  ✓ 統計・分析機能（グラフ表示）
  ✓ データバックアップ・復元
  ✓ CSV出力（Excel対応）
  ✓ オフライン動作
  ✓ 利用者管理

🆘 トラブルシューティング
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▼ インストールが失敗する
  → 管理者権限で実行してみてください
  → ウイルス対策ソフトを一時的に無効化してみてください

▼ アプリが起動しない
  → PCを再起動してみてください
  → インストールをやり直してみてください

▼ データが表示されない
  → アプリを再起動してみてください
  → データ管理パネルから「強制リロード」を実行してください
