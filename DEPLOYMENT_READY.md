## 🚀 次のステップ

### 即座に実行可能

1. **依存関係インストール**
   ```bash
   npm install
   ```

2. **Electronアプリビルド**
   ```bash
   # Windows向け
   npm run electron:build:win
   
   # macOS向け（macOS環境が必要）
   npm run electron:build:mac
   
   # Linux向け（Linux環境が必要）
   npm run electron:build:linux
   ```

3. **GitHub Releaseの作成**
   ```bash
   # バージョンアップ＆リリース
   npm run release:patch  # v2.1.0 → v2.1.1
   npm run release:minor  # v2.1.0 → v2.2.0
   ```
