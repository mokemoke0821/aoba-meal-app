# あおば給食アプリ 操作マニュアル（導入・基本編）

## はじめに

このマニュアルは、あおば給食アプリを初めて使う方向けの基本操作ガイドです。  
専門的な知識は不要です。この手順に従って操作してください。

---

## 1. アプリの開き方

### 1.1 ブラウザで開く

1. パソコンやタブレットのブラウザ（Chrome、Edge、Safariなど）を開きます
2. あおば給食アプリのURLを入力してアクセスします
3. 初回アクセス時は、少し時間がかかることがあります

[ここにアプリのトップ画面の画像を貼る]

### 1.2 ホーム画面に追加する（おすすめ）

アプリをホーム画面に追加すると、次回から簡単に開けます。

**パソコンの場合（Chrome/Edge）:**
1. ブラウザのアドレスバー右側の「インストール」アイコンをクリック
2. 「インストール」ボタンをクリック
3. デスクトップにショートカットが作成されます

**タブレット・スマートフォンの場合:**
1. 画面下部に表示される「インストール」ボタンをタップ
2. ホーム画面に追加されます

[ここにインストールプロンプトの画像を貼る]

**メリット:**
- オフラインでも使えます（インターネット接続がなくても動作します）
- アプリのように簡単に起動できます
- データは端末内に保存されます

## 2. 利用者の登録方法

### 2.1 一括登録（おすすめ・新機能）

**名簿をコピーして貼り付けるだけで、複数の利用者を一度に登録できます！**

#### 手順

1. **管理者画面を開く**
   - 画面右上の歯車アイコン（⚙️）をクリック
   - パスコードが設定されている場合は、4桁の数字を入力します

2. **「ユーザー管理」を選択**
   - 管理者メニュー画面で「ユーザー管理」カードをクリック

3. **「一括登録」パネルを開く**
   - 画面上部の「一括登録」をクリックして展開します

[ここに一括登録パネルの画像を貼る]

4. **グループを選択**
   - 「グループ」のドロップダウンから、登録する利用者のグループを選択します
   - 例：A型利用者、B型利用者、職員、体験利用者

5. **利用者名を入力**
   - 大きなテキストボックスに、利用者名を1行に1名ずつ入力します
   - **名簿からコピー＆ペーストできます！**
   
   **入力例:**
   ```
   山田 太郎
   鈴木 花子
   佐藤 次郎
   高橋 一郎
   ```

6. **「登録する」ボタンをクリック**
   - 入力した利用者が一括で登録されます
   - 既に登録されている名前は自動的にスキップされます
   - 登録結果が画面下部に表示されます

**結果メッセージの見方:**
- 「○件の利用者を追加しました」→ すべて新規登録されました
- 「○件の利用者を追加しました（重複スキップ: ○件）」→ 一部は既に登録済みでした

**注意点:**
- 空行は自動的に無視されます
- 同じ名前の利用者は登録できません（重複チェック）
- 1行に1名ずつ入力してください

### 2.2 1名ずつ登録する方法

1. **管理者画面を開く**
   - 画面右上の歯車アイコン（⚙️）をクリック

2. **「ユーザー管理」を選択**

3. **「新規登録」ボタンをクリック**
   - 画面上部の「新規登録」ボタンをクリック

4. **利用者情報を入力**
   - 利用者名、グループ、料金などを入力します
   - 「登録」ボタンをクリック

## 3. 日々の記録方法

### 3.1 給食の注文を記録する

1. **ホーム画面でカテゴリを選択**
   - A型利用者、B型利用者、職員、体験利用者のいずれかを選択

2. **利用者を選択**
   - 利用者一覧から、給食を注文する利用者を選択

3. **注文を確定**
   - 「注文する」ボタンをクリック

### 3.2 食べた量を記録する（10段階評価）

給食を食べ終わった後、食べた量を記録します。

1. **利用者を選択**
   - ホーム画面から利用者を選択

2. **「食べた量を記録」を選択**

3. **10段階で評価**
   - 1割～10割まで、食べた量を選択します
   - **10割 = 完食**です

[ここに食べた量記録画面の画像を貼る]

**評価の目安:**
- 1割：ほとんど食べられなかった
- 5割：半分くらい食べた
- 10割：完食した

4. **「記録する」ボタンをクリック**
   - 記録が保存されます

### 3.3 支援記録を追加する（任意）

食べた量を記録する際に、「支援記録・備考」欄にメモを残すことができます。

- 例：「スプーンで支援した」「おかわりした」など

## 4. よく使う機能

### 4.1 利用者を検索する

利用者管理画面で、画面上部の検索ボックスに名前を入力すると、該当する利用者が表示されます。

### 4.2 グループで絞り込む

利用者管理画面で、「グループ」のドロップダウンからグループを選択すると、そのグループの利用者のみが表示されます。

---

## 5. トラブルシューティング

### アプリが開かない

- ブラウザを再起動してみてください
- 別のブラウザで試してみてください

### データが消えた

- ブラウザの履歴やキャッシュを削除すると、データが消えることがあります
- 定期的にバックアップを取ることをおすすめします（管理者向けマニュアル参照）

### 画面が表示されない

- インターネット接続を確認してください
- ただし、一度開いたアプリはオフラインでも使えます
