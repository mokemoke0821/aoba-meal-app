# データ管理ガイド

このアプリはインターネット上にデータを保存せず、お使いの端末（タブレットやPC）の中にデータを保存します。
そのため、定期的なバックアップが非常に重要です。

## バックアップの取り方（推奨：週1回）
1. 「管理者パネル」→「データ管理」を選択
2. 「バックアップ作成」ボタンを押す
3. `aoba-meal-backup-日付.json` というファイルがダウンロードされます
4. このファイルをUSBメモリやGoogleドライブなどに保存してください

## データの復元（機種変更時など）
1. 新しい端末でアプリを開く
2. 「管理者パネル」→「データ管理」を選択
3. 「バックアップから復元」の「ファイルを選択」ボタンを押す
4. 保存しておいたバックアップファイル（.json）を選択
5. 「復元実行」ボタンを押す
   - **注意**: 現在のデータは上書きされます

## データの共有
複数の端末で同じデータを使いたい場合は、以下の手順で行います。
1. 元の端末で「バックアップ作成」を行う
2. 作成されたファイルをUSBメモリなどで別の端末に移動
3. 別の端末で「バックアップから復元」を行う

**注意**: 自動同期機能はありません。手動での受け渡しが必要です。
