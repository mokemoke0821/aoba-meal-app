# あおば給食アプリ よくある質問（FAQ）

## はじめに

このページでは、あおば給食アプリを使う上でよくある質問とその回答をまとめています。  
困ったときは、まずこちらをご確認ください。

---

## 基本操作について

### Q1. アプリを開くにはどうすればいいですか？

**A:** ブラウザでアプリのURLにアクセスするか、ホーム画面に追加したショートカットから開きます。

詳しくは「01_QuickStart.md」の「1. アプリの開き方」をご覧ください。

### Q2. ネットがなくても使えますか？

**A:** はい、使えます。

一度アプリを開いておけば、インターネット接続がなくても使えます（オフライン動作）。  
ただし、初回アクセス時やアプリの更新時は、インターネット接続が必要です。

ホーム画面に追加しておくと、より確実にオフラインで使えます。

### Q3. スマートフォンやタブレットでも使えますか？

**A:** はい、使えます。

パソコン、タブレット、スマートフォンのどの端末でも、ブラウザがあれば使えます。  
タブレットでの操作を想定して設計されているため、タッチ操作でも快適に使えます。

## 利用者登録について

### Q4. 複数の利用者を一度に登録できますか？

**A:** はい、できます！（新機能）

「一括登録」機能を使うと、名簿をコピー＆ペーストするだけで、複数の利用者を一度に登録できます。

**手順:**
1. 管理者画面 → 「ユーザー管理」
2. 「一括登録」パネルを開く
3. グループを選択
4. 利用者名を1行に1名ずつ入力（名簿からコピー＆ペースト可）
5. 「登録する」ボタンをクリック

詳しくは「01_QuickStart.md」の「2.1 一括登録」をご覧ください。

### Q5. 同姓同名の人がいる場合はどうすればいいですか？

**A:** 同じ名前の利用者は登録できません。名前を変えて登録してください。

**例:**
- 山田太郎 → 山田太郎A、山田太郎B
- 佐藤花子 → 佐藤花子（1組）、佐藤花子（2組）
- 鈴木一郎 → 鈴木一郎（父）、鈴木一郎（子）

名前の後ろに識別できる情報を追加して登録してください。

### Q6. 利用者名の長さに制限はありますか？

**A:** はい、最大50文字までです。

50文字を超える名前は登録できません。短くするか、別の名前で登録してください。

### Q7. 一括登録で「重複スキップ」と表示されました。どういう意味ですか？

**A:** 入力した名前のうち、既に登録されている名前は自動的にスキップ（無視）されたという意味です。

例えば、10名を一括登録しようとして、3名が既に登録済みだった場合：
- 「7件の利用者を追加しました（重複スキップ: 3件）」と表示されます
- 新規の7名のみが登録され、既存の3名はそのまま残ります

これは正常な動作です。重複を防ぐための機能です。

## パスコード・セキュリティについて

### Q8. パスコードを忘れました。どうすればいいですか？

⚠️ **重要：パスコードを忘れた場合の対処方法**

パスコードを忘れた場合、ブラウザのLocalStorageをクリアすることでリセットできますが、**すべてのデータも消えてしまいます**。

**対処方法:**

1. **バックアップを取っている場合:**
   - ブラウザのLocalStorageをクリア（データが消えます）
   - バックアップファイルからデータを復元
   - 新しいパスコードを設定

2. **バックアップを取っていない場合:**
   - データを失うことになります
   - ブラウザのLocalStorageをクリア
   - 新しいパスコードを設定
   - 利用者を再登録

⚠️ **パスコードを忘れないようにメモしておいてください。また、定期的にバックアップを取ることをおすすめします。**

**ブラウザのLocalStorageをクリアする方法（Chrome/Edge）:**
1. ブラウザの設定を開く
2. 「プライバシーとセキュリティ」→「閲覧履歴データの削除」
3. 「Cookie と他のサイトデータ」にチェック
4. 「データを削除」をクリック

### Q9. パスコードは設定した方がいいですか？

**A:** はい、設定することを強く推奨します。

パスコードを設定すると、誰でも管理者画面に入れなくなります。  
誤操作や不正アクセスを防ぐため、必ず設定してください。

**設定方法:**
1. 管理者画面 → 「セキュリティ設定」セクション
2. 「管理者パスコードを有効にする」をON
3. 4桁の数字を入力・確認
4. 「設定」ボタンをクリック

詳しくは「02_AdminGuide.md」の「1.2 パスコード認証」をご覧ください。

### Q10. パスコードは何桁ですか？

**A:** 4桁の数字です。

数字のみで、0000～9999の範囲で設定できます。  
覚えやすい数字を設定してください（例：生年月日の下4桁など）。

## データ管理について

### Q11. データはどこに保存されますか？

**A:** ブラウザのLocalStorageに保存されます。

- データは各端末（パソコン、タブレットなど）に個別に保存されます
- ブラウザの履歴やキャッシュを削除すると、データが消える可能性があります
- 定期的にバックアップを取ることをおすすめします

### Q12. バックアップはどうやって取りますか？

**A:** 管理者画面から手動でバックアップを取るか、自動バックアップを設定できます。

**手動バックアップ:**
1. 管理者メニュー → 「統計・データ管理」
2. 「バックアップ設定」セクション
3. 「手動バックアップ」をクリック
4. JSONファイルがダウンロードされます

**自動バックアップ:**
1. 「バックアップ設定」セクション
2. 「自動バックアップを有効にする」をON
3. バックアップ頻度を選択（5分、10分、30分、1時間）
4. 「保存」ボタンをクリック

詳しくは「02_AdminGuide.md」の「4. バックアップと復元」をご覧ください。

### Q13. データを別の端末に移すにはどうすればいいですか？

**A:** バックアップファイルを使って移行できます。

1. **元の端末でバックアップを取る**
   - 管理者画面 → 「統計・データ管理」→ 「手動バックアップ」

2. **バックアップファイルを新しい端末にコピー**
   - USBメモリ、メール、クラウドストレージなどを使って転送

3. **新しい端末でデータを復元**
   - 管理者画面 → 「統計・データ管理」→ 「データ復元」
   - バックアップファイルを選択して復元

### Q14. データが消えてしまいました。復元できますか？

**A:** バックアップファイルがあれば復元できます。

1. 管理者画面 → 「統計・データ管理」
2. 「データ復元」セクション
3. 以前に保存したバックアップファイルを選択
4. 「復元する」ボタンをクリック

⚠️ **バックアップファイルがない場合は、データを復元できません。定期的にバックアップを取ることをおすすめします。**

## 操作・機能について

### Q15. 利用者を削除するにはどうすればいいですか？

**A:** 1名ずつ削除する方法と、複数名を一括削除する方法があります。

**1名ずつ削除:**
1. 利用者管理画面で、削除したい利用者の行の「削除」アイコンをクリック
2. 確認ダイアログで「OK」をクリック

**複数名を一括削除（新機能）:**
1. 利用者一覧の左側のチェックボックスで、削除したい利用者を選択
2. 「選択したユーザーを削除」ボタン（赤色）をクリック
3. 確認ダイアログで「削除する」をクリック

⚠️ **削除したデータは復元できません。削除前にバックアップを取ることをおすすめします。**

詳しくは「02_AdminGuide.md」の「2.3 利用者の削除」をご覧ください。

### Q16. CSVファイルを出力するにはどうすればいいですか？

**A:** 利用者一覧や統計データをCSV形式で出力できます。

**利用者一覧のCSV出力:**
1. 利用者管理画面
2. 「エクスポート」ボタンをクリック

**統計データのCSV出力:**
1. 管理者メニュー → 「統計・データ管理」
2. 出力したいデータの種類を選択（統計CSV、月次CSV、期間指定CSVなど）
3. 「CSV出力」ボタンをクリック

出力されたCSVファイルは、Excelなどで開いて確認できます。

詳しくは「02_AdminGuide.md」の「3. データの出力（CSV）」をご覧ください。

### Q17. 食べた量の記録はどうやってしますか？

**A:** 10段階（1割～10割）で記録します。

1. ホーム画面から利用者を選択
2. 「食べた量を記録」を選択
3. 1割～10割のいずれかを選択（10割 = 完食）
4. 「記録する」ボタンをクリック

詳しくは「01_QuickStart.md」の「3.2 食べた量を記録する」をご覧ください。

## トラブルシューティング

### Q18. アプリが開きません

**A:** 以下の方法を試してください。

1. ブラウザを再起動する
2. 別のブラウザで試す（Chrome、Edge、Safariなど）
3. インターネット接続を確認する
4. ブラウザのキャッシュをクリアする

### Q19. 画面が正しく表示されません

**A:** 以下の方法を試してください。

1. ブラウザを最新版に更新する
2. ブラウザのキャッシュをクリアする
3. 別のブラウザで試す

### Q20. データが保存されません

**A:** ブラウザの設定を確認してください。

- ブラウザのCookieやLocalStorageが無効になっていないか確認
- シークレットモード（プライベートモード）では、データが保存されない場合があります
- 通常モードで使用してください

### Q21. エラーメッセージが表示されます

**A:** エラーメッセージの内容を確認してください。

- よくあるエラーは、入力内容が正しくない場合です（例：必須項目が空、文字数制限超過など）
- エラーメッセージに従って、入力内容を修正してください
- 解決しない場合は、管理者に相談してください

## その他

### Q22. 複数の端末で同じデータを使いたいです

**A:** 現在のバージョンでは、端末間でデータを自動的に同期する機能はありません。

各端末に個別にデータが保存されます。  
データを共有する場合は、バックアップファイルを使って手動で移行してください。

### Q23. 利用者を編集できますか？

**A:** はい、編集できます。

1. 利用者管理画面で、編集したい利用者の行の「編集」アイコンをクリック
2. 利用者情報を変更
3. 「更新」ボタンをクリック

詳しくは「02_AdminGuide.md」の「2.2 利用者の編集」をご覧ください。

### Q24. 利用者を一時的に無効化できますか？

**A:** はい、できます。

利用者を削除せずに、一時的に無効化できます。

1. 利用者管理画面で、該当利用者の行の「無効化」アイコンをクリック
2. 無効化された利用者は、通常の利用者一覧に表示されなくなります
3. 「無効ユーザーも表示」スイッチをONにすると、無効化された利用者も表示されます
