# クイックスタートガイド

## 1. アプリへのアクセス
タブレットまたはPCのブラウザ（Chrome推奨）で以下のURLにアクセスしてください。

**https://mokemoke0821.github.io/aoba-meal-app**

## 2. ホーム画面への追加（アプリのインストール）
インターネットがない場所でも使えるように、アプリをホーム画面に追加します。

### タブレット（Android）の場合
1. 画面右上のメニュー（︙）をタップ
2. 「ホーム画面に追加」または「アプリをインストール」を選択
3. 「追加」または「インストール」をタップ

### iPad（iOS）の場合
1. 画面右上の共有ボタン（□から↑が出ているアイコン）をタップ
2. 「ホーム画面に追加」を選択
3. 「追加」をタップ

## 3. 初回設定
1. ホーム画面に追加されたアイコンをタップして起動
2. 利用者が0件の状態から始まります
3. 「管理者パネル」→「利用者管理」から利用者を登録してください

これで準備完了です！
