# 基本操作マニュアル

## 利用者の登録・編集
1. 画面右上の「管理者パネル」をクリック（パスワード: `admin`）
2. 「利用者管理」タブを選択
3. **新規登録**: 「新規登録」ボタンを押し、名前・グループ・カテゴリを入力して保存
4. **編集**: 一覧の「編集」ボタンから内容を変更
5. **削除**: 「削除」ボタンで利用者を削除（※過去の記録は残ります）

## 給食の注文（毎日の操作）
1. トップ画面で利用者のカテゴリ（A型/B型など）を選択
2. 利用者の名前を選択
3. 「注文する」ボタンを押す
4. 確認画面で「はい」を選択

## 摂食量の入力（食後の操作）
1. 注文済みの利用者を選択
2. 「評価を入力」画面が表示されます
3. 食べた量を選択（1〜10の数字）
   - 10: 完食
   - 5: 半分
   - 1: 一口程度
4. 必要に応じて「支援記録」にメモを入力
5. 「評価を送信」を押す

## データの確認
1. 「管理者パネル」→「統計・分析」を選択
2. 日付を選択すると、その日の注文状況がグラフで表示されます
3. 「月次レポート」タブで月ごとの集計を確認できます

## CSV出力（月末の集計）
1. 「管理者パネル」→「データ管理」を選択
2. 「CSVエクスポート」セクションへ移動
3. 対象月を選択して「ダウンロード」ボタンを押す
4. エクセル等で開けるファイルが保存されます
