# 📋 UAT（ユーザー受け入れテスト）ガイド

## あおば給食管理アプリ Electron版 v2.1.0

---

## 📌 このドキュメントについて

このガイドは、あおば給食管理アプリの**ユーザー受け入れテスト（UAT: User Acceptance Testing）**を実施するための手順書です。

### UAT の目的

1. **実際の利用環境**でアプリが正しく動作することを確認
2. **エンドユーザー視点**で使いやすさを評価
3. **業務フロー**に沿った動作を検証
4. **問題点**を発見し、リリース前に修正

---

## 👥 テスト参加者

### 推奨構成

| 役割 | 人数 | 要件 |
|------|------|------|
| **現場職員** | 2-3名 | 実際に給食記録を担当する職員 |
| **管理者** | 1名 | データ管理・統計機能を使用する責任者 |
| **IT担当** | 1名 | 技術的な問題に対応できる人 |

### テスト実施日数

- **期間**: 3-5日間
- **1日あたりテスト時間**: 30分-1時間

---

## 🖥️ テスト環境準備

### 必要な機器

- [ ] テスト用PC（Windows 10/11）
- [ ] タブレット（あれば）
- [ ] インターネット接続（初回のみ、オフラインテストのため）

### インストール手順

1. **インストーラーのダウンロード**
   - [GitHub Releases ページ](https://github.com/mokemoke0821/aoba-meal-app/releases) から最新版をダウンロード
   - ファイル: `あおば給食管理-Setup-2.1.0.exe`

2. **インストール実行**
   - ダウンロードしたファイルをダブルクリック
   - 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」
   - インストールウィザードに従って進む

3. **起動確認**
   - デスクトップのショートカットから起動
   - または、スタートメニューから「あおば給食管理」を検索

## 📝 テストシナリオ

### シナリオ 1: 初回起動と設定

#### 目的
アプリが正常にインストールされ、初期画面が表示されることを確認

#### 手順
1. デスクトップショートカットをダブルクリック
2. アプリが起動するまでの時間を計測（⏱️ 目標: 3秒以内）
3. カテゴリ選択画面が表示されることを確認

#### 確認ポイント
- [ ] アプリが3秒以内に起動した
- [ ] カテゴリ選択画面が表示された
- [ ] 4つのカテゴリ（A型、B型、職員、体験者）が表示された
- [ ] 各カテゴリが色分けされている

#### 問題があった場合
→ **フィードバックフォーム**に記入してください

---

### シナリオ 2: 摂食量記録（標準フロー）

#### 目的
給食の摂食量を記録する基本的な操作が正しく動作することを確認

#### 手順
1. **カテゴリ選択**: 「A型」をクリック
2. **利用者選択**: 「1 田中太郎」をクリック（架空の利用者）
3. **給食希望確認**: 「はい」をクリック
4. **摂食量記録**: 「8割程度」をクリック
5. **支援記録入力**: 「完食。食欲旺盛でした。」と入力
6. **記録ボタン**: 「記録する」をクリック
7. カテゴリ選択画面に戻ることを確認

#### 確認ポイント
- [ ] 各画面がスムーズに遷移した
- [ ] 利用者情報が正しく表示された
- [ ] 摂食量ボタンが選択できた
- [ ] 支援記録が入力できた
- [ ] 記録が保存された（次のステップで確認）

#### スクリーンショット
📸 各画面のスクリーンショットを撮影してください

---

### シナリオ 3: 記録の確認（統計画面）

#### 目的
記録したデータが正しく保存され、統計画面で確認できることを検証

#### 手順
1. 右下の「管理者」ボタンをクリック
2. **今日の状況**を確認
   - 本日の記録数が「1件」になっているか
   - 平均摂食量が「8.0割」と表示されているか
3. **記録待ち利用者リスト**を確認
   - 先ほど記録した利用者が消えているか
4. **摂食量分布グラフ**を確認
   - 8割の列が1件になっているか

#### 確認ポイント
- [ ] 今日の状況に反映されている
- [ ] 記録待ち利用者リストが更新された
- [ ] グラフが表示されている
- [ ] データが正確

---

### シナリオ 4: 複数利用者の記録

#### 目的
複数の利用者のデータを連続して記録できることを確認

#### 手順
1. カテゴリ選択画面に戻る（左上の「戻る」ボタン）
2. 以下の利用者を順番に記録：
   - **A型 2番**: 10割（完食）
   - **B型 1番**: 5割程度
   - **職員 1番**: 9割程度
3. 各記録後、統計画面で件数が増えることを確認

#### 確認ポイント
- [ ] 4件すべてが記録できた
- [ ] 統計画面で「本日の記録数: 4件」と表示
- [ ] 平均摂食量が正しく計算されている（(8+10+5+9)/4 = 8.0割）

---

### シナリオ 5: データ管理（バックアップ・復元）

#### 目的
データのバックアップと復元が正しく機能することを確認

#### 手順
1. 統計画面の**データ管理パネル**を開く
2. 「JSONバックアップを作成」をクリック
3. ファイルが保存されることを確認（ダウンロードフォルダ）
4. 「全データを削除」をクリック（⚠️ 注意: テストデータのみ）
5. 削除確認ダイアログで「はい」をクリック
6. カテゴリ選択画面に戻る
7. 統計画面で記録数が0件になっていることを確認
8. 「JSONバックアップを復元」をクリック
9. 先ほど保存したバックアップファイルを選択
10. データが復元されることを確認

#### 確認ポイント
- [ ] バックアップファイルが作成された
- [ ] ファイル名が「あおば給食データバックアップ_YYYYMMDD.json」形式
- [ ] データ削除が実行された
- [ ] バックアップから復元できた
- [ ] 復元後のデータ件数が一致

---

### シナリオ 6: CSV出力

#### 目的
データをCSV形式でエクスポートできることを確認

#### 手順
1. 統計画面のデータ管理パネルを開く
2. 「利用者一覧をCSV出力」をクリック
3. ファイルが保存されることを確認
4. 「給食記録をCSV出力」をクリック
5. ファイルが保存されることを確認
6. Excelで両方のCSVファイルを開く

#### 確認ポイント
- [ ] 利用者一覧CSVが出力された
- [ ] 給食記録CSVが出力された
- [ ] Excelで開いたとき、文字化けしていない
- [ ] データが正しく表示されている
- [ ] ヘッダー行（項目名）が日本語で表示されている

---

### シナリオ 7: オフライン動作

#### 目的
インターネット接続がなくてもアプリが動作することを確認

#### 手順
1. PCのWi-Fiをオフにする（または、LANケーブルを抜く）
2. アプリを再起動
3. 摂食量記録を1件追加
4. 統計画面で確認
5. バックアップを作成

#### 確認ポイント
- [ ] オフライン状態でもアプリが起動した
- [ ] 記録が正常に保存された
- [ ] 統計画面が表示された
- [ ] バックアップが作成できた

---

### シナリオ 8: 長期間使用シミュレーション

#### 目的
1週間～1ヶ月分のデータを扱った場合のパフォーマンスを確認

#### 手順
1. テストデータジェネレーター（別途提供）を使用して、100件のダミーデータを作成
2. JSONバックアップとして復元
3. 統計画面を開く
4. グラフが正しく表示されることを確認
5. フィルタリング（日別/週別/月別）を試す

#### 確認ポイント
- [ ] 100件のデータが読み込めた
- [ ] 画面が重くなっていない
- [ ] グラフがスムーズに表示された
- [ ] フィルタリングが正常に動作した

---

### シナリオ 9: ウィンドウサイズ変更

#### 目的
ウィンドウサイズを変更しても、レイアウトが崩れないことを確認

#### 手順
1. アプリのウィンドウを最小化（800x600）
2. 各画面を確認
3. ウィンドウを最大化
4. 各画面を確認
5. ウィンドウを中間サイズ（1280x720）に調整
6. 各画面を確認

#### 確認ポイント
- [ ] 最小サイズでもレイアウトが崩れない
- [ ] ボタンやテキストが見切れていない
- [ ] グラフが適切にリサイズされる
- [ ] スクロールバーが必要な場合のみ表示される

---

### シナリオ 10: アンインストール

#### 目的
アプリが正しくアンインストールできることを確認

#### 手順
1. Windowsの「設定」→「アプリ」→「インストールされているアプリ」を開く
2. 「あおば給食管理」を検索
3. 「アンインストール」をクリック
4. アンインストールウィザードに従う
5. デスクトップショートカットが削除されたことを確認
6. `C:\Program Files\あおば給食管理` または `%LOCALAPPDATA%\Programs\あおば給食管理` フォルダが削除されたことを確認

#### 確認ポイント
- [ ] アンインストールが完了した
- [ ] ショートカットが削除された
- [ ] アプリフォルダが削除された
- [ ] スタートメニューから消えた
