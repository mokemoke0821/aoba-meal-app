## [2.0.0] - 2025-10-07

### Added
- ⏳ **記録待ち利用者リスト機能**: 本日未記録の利用者を一覧表示
- 📊 **統計パネル強化**: カテゴリ別グループ化表示
- 💾 **自動バックアップ機能**: 7日ごとの自動バックアップ
- 📤 **CSV出力機能**: 利用者一覧と給食記録のCSV出力
- 📥 **CSV読み込み機能**: CSVからの一括データインポート
- 🔧 **データ診断ツール**: LocalStorageの状態確認
- 🌐 **PWA完全対応**: オフライン動作、インストール可能

### Changed
- 🎨 **UI改善**: Material-UI v7.1.2 にアップグレード
- ⚡ **パフォーマンス最適化**: ビルドサイズ削減
- 📱 **レスポンシブデザイン**: タブレット/スマホ対応強化

### Fixed
- 🐛 **LocalStorage同期問題**: 保存タイミングの最適化
- 🔄 **画面遷移バグ**: 戻るボタンの動作を修正
- 📊 **統計計算エラー**: 摂食量計算ロジックを修正

## [1.0.0] - 2024-12-01

### Added
- 🎉 **初回リリース**: 就労移行支援事業所向け給食摂食量管理システム
- 🍽️ **摂食量記録機能**: 1割～10割の10段階評価
- 👥 **利用者カテゴリ管理**: A型/B型/職員/体験者
- 📊 **統計・分析機能**: 摂食量分布グラフ、日別注文数推移
- 💾 **データ永続化**: LocalStorageによる自動保存
- 🎨 **Material-UI採用**: モダンなUIデザイン
- 📱 **レスポンシブデザイン**: PC/タブレット/スマホ対応
- 🌐 **GitHub Pages デプロイ**: 本番環境公開

### Technical
- ⚛️ **React 19.1.0**: 最新のReactを採用
- 🔷 **TypeScript 4.9.5**: 完全な型安全性
- 📦 **Create React App + Craco**: ビルド環境
- 🧪 **Jest + React Testing Library**: テストカバレッジ 94.93%
- 📈 **Recharts**: グラフ可視化ライブラリ
- 🎨 **Material-UI v7**: UIコンポーネントライブラリ
